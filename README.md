# Crypto Trading Bot

A Python crypto trading bot using Binance API with technical indicators.

## Features

- EMA strategy
- RSI indicator
- MACD crossover
- Telegram alerts
- Backtesting
- Chart plotting
- Risk management

## Tech Stack

Python
CCXT
Pandas
TA-Lib
Matplotlib

## Installation

pip install -r requirements.txt

## Run Bot

python bot.py

## Configuration

Edit the `.env` file:

BINANCE_API_KEY=your_key
BINANCE_SECRET_KEY=your_secret
TELEGRAM_TOKEN=your_token
CHAT_ID=your_chat_id
